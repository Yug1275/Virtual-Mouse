# Virtual Mouse using Hand Gestures

Control your computer mouse using hand gestures via webcam.

## Tech Stack
- Python, OpenCV, MediaPipe, PyAutoGUI, NumPy

## Setup
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python src/main.py
```
